# StoryBored Studios - Official Landing Page

This is the central landing page for **StoryBored Studios**, a premium game development collective. 

## 🚀 Getting Started

To run the development server locally:

```bash
# Navigate to the project directory
cd StoryBoredStudios

# Install dependencies (if not already done)
npm install

# Start the dev server
npm run dev
```

## 🏗️ Architecture

- **Framework**: [Vite](https://vitejs.dev/) + [React](https://reactjs.org/)
- **Styling**: Vanilla CSS with modern Flexbox/Grid and Glassmorphism.
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🎨 Design Philosophy

- **Premium Aesthetics**: High-energy gradients, subtle blurs, and glassmorphism.
- **Micro-Animations**: Smooth transitions and hover effects to create a "living" interface.
- **Mobile First**: Fully responsive design that scales from desktop to smartphone seamlessly.

## 📁 Project Structure

- `src/App.jsx`: Main landing page orchestrator.
- `src/index.css`: Global design system and theme tokens.
- `src/assets/`: Visual assets and brand elements.
